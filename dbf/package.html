Provides classes to access data from a DBF file.

DBF is a somehow obsolete file format used by Microsoft Visual Fox Pro.
Its specifications can be found at:
http://msdn.microsoft.com/en-us/library/aa977077%28VS.71%29.aspx
This package does not aim at fully implementing the specification.
It can only read fields of type:
C - Character
M - Memo
N - Numeric
In order to read fields of type Memo the DBF class relies on the FPT class.
